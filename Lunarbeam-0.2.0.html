What’s new in 0.2.0

  • Builds, saves, and publishing now show clear errors instead of failing silently
  • Large projects save in the background without freezing the canvas
  • More reliable publishing to Vercel, with automatic retries on network hiccups
  • The live preview recovers from start-up problems instead of showing a blank page
  • New Settings window: use your own Pexels API key and manage anonymous usage sharing
  • Stability and performance improvements across the editor
