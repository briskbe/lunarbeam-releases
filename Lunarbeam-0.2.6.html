What’s new in 0.2.6

  • New: Design Copilot keeps the selected target, direct controls, quick actions, and Lunar Agent together in one focused dock.
  • New: Review mode now has a clear comment workflow with numbered pins, a live context bar, Fix All, and Done.
  • Improved: Design selections now use clearer element and shared-component states, a brief canvas tether, and calmer one-time motion.
  • Improved: Agent progress, questions, and finished changes stay visible in Design mode with immediate Keep or Undo actions.
  • Improved: The entire editor has a more polished, consistent light and dark appearance with stronger contrast and responsive controls.
  • Fixed: Startup recovery no longer presents project trust UI during SwiftUI state initialization.
